# Rock, Paper, Scissors Game

A classic implementation of the Rock, Paper, Scissors game, available in both a browser console version and a modern graphical user interface (GUI) version.

## üöÄ Getting Started

This project is split into two primary branches, each housing a complete version of the game.

### Prerequisites

Ensure you have the following installed on your system:

* **Chrome**
* **Node** 

### 1. Console Version

The console version is a lightweight and quick way to play directly from your browser terminal.

* **Branch:** `main`
* **Best for:** Quick play, scripting, and environments without graphical support.

#### Running the Console Game

1.  **Clone the Repository:**
    ```bash
    git clone git@github.com:thejoshuadibs/rps.git
    ```
2.  **Switch to the Console Branch:**
    ```bash
    cd rps
    ```
3.  **Run the Game:**
    ```bash
    Open with Chrome and open the console 
    ```

### 2. UI Version (GUI/Web)

The UI version provides a visually appealing and interactive experience.

* **Branch:** `rps-ui`
* **Best for:** Desktop users, a richer user experience, and showcasing UI skills.

#### Running the UI Game

1.  **Switch to the UI Branch:**
    ```bash
    git checkout rps-ui
    ```
2.  **Run the Game:**
    ```bash
    Open with Live Preview or Live Server
    ```

## üéÆ Game Features

* **Classic Rules:** Implements standard Rock, Paper, Scissors rules (Rock beats Scissors, Scissors beats Paper, Paper beats Rock).
* **Computer Opponent:** Play against an intelligent computer player by the name of Mo.
* **Score Tracking:** Tracks the score over five rounds.

| Feature | Console (`console-game`) | UI (`ui-game`) |
| :--- | :---: | :---: |
| **Input Method** | Keyboard (text prompts) | Buttons/Clicks |
| **Visuals** | Text-based output | Graphical interface (buttons, images) |
| **Game Speed** | High (minimal rendering overhead) | Standard |
